Add this code for centered text:

<hd style="text-align: center;">text</hd>


Use this code for adding the font:

<head><link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Comic+Relief:wght@400;700&display=swap" rel="stylesheet">
<style>body {
  font-family: 'Comic Relief', sans-serif;
}
</style></head>

Use this code for making links:

<a href="Link" style="text-decoration:none; font-size: 175%;">Link Text</a>


Use this code to make text bigger:

style="font-size: num%;"


Use this code for automatic dark mode:

<meta name="color-scheme" content="light dark" />
