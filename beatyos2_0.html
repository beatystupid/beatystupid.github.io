BeatyOS 2.0 is Released!

BeatyOS has been released for real
hard drives and VirtualBox! If you want to
install it in VirtualBox, download the
BeatyOS 2.0 VirtualBox 7-Zip, export it,
and put it in your VirtualBox VMs folder.
For installing on real hard drives, download
the real-drive 7-Zip, export it, and then
import it into your hard drive. I recommend
importing with Visual Disk Imager. Have fun
with BeatyOS 2.0!


Download BeatyOS 2.0 VirtualBox 7-Zip

(The 7-Zip file size is 6.2 Gigabytes.)
(The extracted file size is 17.4 Gigabytes)

(The 7-Zip file size is 6.27 Gigabytes)
