BeatyOS 2.0 is Released!

BeatyOS 2.0 has FINALLY been uploaded and released! It is
compressed as a 7-Zip, so to get it, make sure you
have 7-Zip to extract it. For VirtualBox, place the folder in your VMs folder.
For actual computers, import the .vhd file to your hard drive.
